# Doceria da Bruna 🧁

E-commerce para uma doceria: catálogo com foto e preço, pedidos com pagamento na
entrega, frete por região, cadastro opcional com desconto e painel de gestão para a
dona. Identidade visual em tons pastéis, delicada, com um versículo bíblico discreto.
